# py_awake

## Overview
This repository contains two Python scripts designed to manage system sleep behavior:
1. **keep_awake.py**: A script that maintains the system awake state using various approaches.
2. **keep_awake_minimal.py**: A minimalistic version of the keep_awake script with essential functionality.

## Scripts Documentation

### keep_awake.py
- **Purpose**: This script prevents your computer from going to sleep by running specific operations that keep the system active.
- **Usage Instructions**:
  - Run the script in your terminal with the command:  
    `python keep_awake.py`
- **Parameters**: It may take parameters to customize its behavior (refer to the code comments for more details).
- **Requirements**: Ensure you have Python installed on your system.

### keep_awake_minimal.py
- **Purpose**: A stripped-down version of the keep_awake script, which keeps the system awake with fewer resources.
- **Usage Instructions**:
  - Execute it with:  
    `python keep_awake_minimal.py`
- **Parameters**: Minimal parameters required (check code for usage).
- **Requirements**: Same as above, a Python environment is needed.

## Conclusion
Choose the script that best fits your needs for keeping your system active, whether you require full functionality or a minimal approach.
